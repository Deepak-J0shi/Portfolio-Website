# Portfolio — Deepak Joshi

![Portfolio Screenshot](public/screenshot.png)

**Live Demo:** https://portfolio-main-tan-three.vercel.app/

A personal portfolio website showcasing projects, skills, and contact information. Built with Next.js, Tailwind CSS and focused on minimal, fast, and accessible presentation for ML / Data Science / Backend projects.

---

## Key features

- Clean, responsive UI (mobile-first)
- Project showcase with links to GitHub repos & demo
- Fast performance using Next.js static / hybrid rendering
- Accessible components and semantic markup
- Easy to update: projects and content driven by MD/JSON (or a simple data file)
- Ready for Vercel / Netlify deployment

---

## Tech stack

- Next.js (React)
- Tailwind CSS
- TypeScript (optional)
- PostCSS
- Vercel (recommended for deployment)
- (Optional) Analytics, sitemap, SEO meta tags

---

## Quick start (development)

> Requires Node.js (v18+) and a package manager (pnpm / npm / yarn). This repo uses `pnpm` in scripts — swap to `npm` or `yarn` if you prefer.

```bash
# 1. clone the repo
git clone https://github.com/JoshiDeepak08/<portfolio-repo-name>.git
cd <portfolio-repo-name>

# 2. install dependencies
pnpm install
# or
# npm install
# yarn install

# 3. run dev server
pnpm dev
# opens at http://localhost:3000
